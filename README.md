CommOAuth2
==========
特性

1.支持国内淘宝，支付宝，新浪，微信，腾讯微博快捷登录

2.支持用消息处理器机制来处理你的业务逻辑

3.支持Session的存取机制(在V3.0后支持)

4.高扩展性,您可以后续根据您的业务进行处理,本库也会对具体平台提供的API进行进一步的整合(预计要在V2.0以后)

5.只要符合OAuth2协议的登录方式都能实现
